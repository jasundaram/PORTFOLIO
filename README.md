Welcome to the source code of my personal portfolio. This website serves as a "Live Quant Sandbox" and an interactive digital resume, showcasing my expertise in data modeling, algorithmic problem-solving, and applied mathematics.

## 🌌 The Aesthetic & UI/UX
The design language of this portfolio breaks away from standard templates. It features a **Neo-Minimalist Dark Mode (#050505)** with sharp neon orange accents (#ff5500), specifically tailored to reflect a high-performance quantitative engineering environment. 

**Key Design Features:**
* **Bento Grid Layout:** For clean, discrete, and highly organized information architecture.
* **Terminal/IDE Windows:** Project sections modeled after code editors and live data feeds.
* **Circuit-Board Timelines:** A chronological node-based UI for educational milestones.
* **Hardware Accelerated Graphics:** Smooth scrolling and custom glitch/decrypt micro-interactions.

## 🛠️ Technical Stack
* **Frontend:** HTML5, Modern CSS (CSS Variables, Flexbox, CSS Grid), Vanilla JS / React (Vite environment).
* **Data Tools Showcased:** Python, SQL, Power BI, DAX, Excel.
* **Mathematical Core:** Probability & Statistics, Numerical Analysis, Number Theory, Applied Cryptography.
